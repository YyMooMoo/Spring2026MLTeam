# Spring2026MLTeam

Welcome to the BAC ML Team repository for Spring 2026!

## Repository Structure

This repository contains various machine learning labs and projects:

- **Lab 1 (`Lab1.py`, `solutions.py`)**: Fundamentals and MNIST dataset operations.
- **Lab 2 (`lab2/`)**: Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN).
- **Lab 4 (`Lab4/`)**: Attention mechanisms.
- **Lab 5 (`lab5/`)**: Additional ML exercises.
- **Lab 6 (`Lab6/`)**: Reinforcement learning with Gridworld environments.

## Datasets

- `mnist_train.csv` and `mnist_test.csv` are included in the root directory for use in training and testing models across different labs.

## Setup

Make sure you have the required Python environments and dependencies installed to run the scripts in each lab directory.
